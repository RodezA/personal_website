## Project Overview

This is a static personal website intended to be hosted on GitHub Pages. It accompanies the resume in `Rodez-Resume.docx`. The site has not yet been implemented.

## Design Requirements

- No emojis anywhere in the site
- Sharp, professional appearance
- Clean and modern design with a simple color scheme
- Sections: About Me, Skills, Experience, Projects, Contact
- Links to LinkedIn and GitHub profiles

## Tech Stack

This is a static site for GitHub Pages. Use plain HTML, CSS, and JavaScript (no build step required) unless there is a specific reason to introduce a static site generator.

## Testing

When validating the site:
- Check all links are functional (LinkedIn, GitHub, any internal links)
- Validate consistent formatting across all sections
- Spell-check all content
