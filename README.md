# Anthony Rodez — Personal Website

Personal portfolio site for Anthony Rodez, Principal AI/Data Engineer.

## Overview

A single-page static site showcasing professional background, skills, experience, and projects. Built with vanilla HTML, CSS, and JavaScript — no frameworks or build tools required.

## Structure

```
├── index.html      # Main page (About, Skills, Experience, Projects, Contact)
├── styles.css      # All styles, including responsive layout
├── script.js       # Mobile nav toggle and scroll behavior
└── Rodez-Resume.docx
```

## Sections

- **About** — Hero summary, education, certifications, and links to LinkedIn and resume
- **Skills** — Technology categories: AI/ML, cloud platforms, data engineering, programming, BI, infrastructure, methodologies
- **Experience** — Roles from 2008 to present (Sparq Consulting, Corellium, Jupiter Intelligence, Panasonic, and others)
- **Projects** — Featured projects with stack details
- **Contact** — Email, phone, LinkedIn, GitHub, and resume link

## Deployment

Hosted via GitHub Pages directly from the `main` branch: https://rodeza.github.io/personal_website/

## Local Development

No build step needed. Open `index.html` in a browser or serve with any static file server:

```bash
python -m http.server 8000
```
